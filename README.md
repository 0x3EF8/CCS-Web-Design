# CCS Department Website

A static front-end website for the **College of Computing Studies (CCS)** department.  
Built with plain HTML, CSS, and JavaScript — no frameworks or build tools required.

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page — hero section, IDE card, quick links |
| `pages/about.html` | About CCS |
| `pages/programs.html` | BSCS & BSIT programs |
| `pages/faculty.html` | Faculty directory |
| `pages/enroll.html` | Enrollment guide |
| `pages/contact.html` | Contact information |

## Assets

| Path | Description |
|------|-------------|
| `assets/css/styles.css` | Global stylesheet |
| `assets/js/main.js` | Navbar, scroll reveal, counters, back-to-top |
| `assets/images/` | Logo and favicon (SVG) |

## Usage

Open `index.html` directly in a browser — no server needed.
